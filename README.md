# Sticks

Универсальная система печати этикеток с ИИ.

## Возможности

- 📋 Создание шаблонов этикеток
- 🏷️ Управление товарами
- 🖨️ Печать этикеток
- 🤖 ИИ модуль для автоматизации
- 📥 Умный парсинг данных
- 🗺️ Карта складов/магазинов

## Платформы

- **GitHub Pages**: https://smol0901-jpg.github.io/sticks/
- **Netlify**: Автоматический деплой
- **Electron**: Настольное приложение

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка
npm run build
```

## Структура


```
sticks/
├── index.html      # Главная страница
├── css/            # Стили
├── js/             # JavaScript модули
└── assets/         # Статические файлы
```
